TJ 第8章 代数编码（Algebraic Coding Theory）

8.1 检错和纠错编码

      一个简单的传输系统模型：
m-digit message $\to$ Encoder $\to$ n-digit code word $\to$ Transmitter $\to$ Noise $\to$ Receiver $\to$ n-digit received word $\to$ Decoder $\to$ m-digit received message or error
      当在传输过程中有一个或很多位被改变了，那么称一个错误出现了。一个译码方案是指一个方法要么将收到的n元组转变位一个有意义的消息，要么给出一个对应n元组的错误信息。
      我们的目标是尽可能便宜且快速的传输没有错误的消息。

奇偶校验

      校验位：偶校验中，校验位保证所有位中1的个数为偶数。
      奇偶校验法有两个缺陷：一是多位错误无法检测，二是无法改正错误。

划区编码？（Block Codes）

      一个编码是一个$(n,m)$-块编码，如果信息可以被划分成$m$二进制位的小块，被一个小块可以被编码称n个二进制位。更进一步的，一个$(n,m)$-块编码由一个编码函数和一个解码函数组成。
      一个码字是指任意一个$E$的值域中的元素。我们也要求$E$是单射的，$D$是满射的。
      令$x=(x_1,\cdots,x_n)$，$y=(y_1,\cdots,y_n)$是二进制n元组。则x和y的汉明距离或距离，$d(x,y)$，是指x和y不同的位的个数。两个码字之间的距离是传输过程中需要把一个码字转换成另一个码字的最少出现的错误次数。一个编码的最小距离，$d_{min}$，是对于所有不同的码字$x,y$，最短的距离$d(x,y)$。码字x的权重，$w(x)$，是指x中1的个数。$w(x)=d(x,0)$。
      命题8.11 令$x,y,z$是二进制n元组，则

  1. $w(x)=d(x,0)$
  2. $d(x,y)\ge0$
  3. $d(x,y)=0$当且仅当$x=y$
  4. $d(x,y)=d(y,x)$
  5. $d(x,y)\le{d(x,z)+d(z,y)}$

      定理8.13 令$C$是一个最小距离$d_{min}=2n+1$的编码，那么$C$可以纠正任何小于等于n位的错，$C$可以检查任何小于等于2n位的错。

8.2 线性编码？（Linear Codes）

      一个群编码（group code)是指一个是$\mathbb{Z}_2^n$子群的编码。
      我们只需要证实一件事以检验一个编码是不是群编码。如果我们将群编码中的任意两个元素相加，所得到的结果一定也是这个编码中的一个n元组。

8.3 监督矩阵和生成矩阵（Parity-Check and Generator Matrices）

      假设$H$是一个$m\times{n}$的矩阵且$n>m$，如果矩阵的后m列构成$m\times{m}$的单位矩阵$I_m$，那么这个矩阵就是一个规范监督矩阵。$H=(A|I_m)$，每一个规范监督矩阵都有一个对应的标准生成矩阵$G=(\frac{I_{n-m}}{A})$。我们的目标是展示$x$满足$Gx=y$当且仅当$Hy=0$。

      定理8.25 如果$H\in\mathbb{M}_{m\times{n}}(\mathbb{Z}_2)$是一个监督矩阵，那么$Null(H)$由所有的满足以下条件的$x\in\mathbb{Z}_2$组成，x的前$n-m$位任意代数最后m位由$Hx=0$决定。最后m位中的每一位都是一个偶校验位，服务于前面$n-m$位中的一些位。因此，$H$决定了一个$(n,n-m)$-块编码。
      以上定理中，x的前$n-m$位称作信息位(information bits)，x的后m位称作校验位(check bits)。

      定理8.26 假设$G$是一个$n\times{k}$的标准生成矩阵，那么$C=\{y:Gx=y$ for $x\in\mathbb{Z}_2^k\}$是一个$(n,k)$-块编码，更进一步的，$C$是一个群编码。

      引理8.27 令$H=(A|I_m)$是一个$m\times{n}$的监督矩阵，$G=(\frac{I_{n-m}}{A})$是一个对应的$n\times(n-m)$的标准生成矩阵，那么$HG=0$。

      定理8.28 令$H=(A|I_m)$是一个$m\times{n}$的监督矩阵，$G=(\frac{I_{n-m}}{A})$是与之对应的$n\times(n-m)$的生成矩阵。令$C$是由$G$生成的一个编码，那么$y$在$C$中当且仅当$Hy=0$。更进一步的，$C$是一个监督矩阵H的线性编码。

      命题8.30 令$e_i$是一个二进制n元组，其中第i位是1，其余位都是0.假设$H\in\mathbb{M}_{m\times{n}}(\mathbb{Z}_2)$，那么$He_i$是矩阵$H$的第i列。
      定理8.31 令$H$是一个$m\times{n}$的二进制矩阵，那么$H$的零空间是一个单错检查编码当且仅当$H$中没有全0列。
      定理8.34 令$H$是一个二进制矩阵，那么$H$的零空间是一个单错纠正编码当且仅当$H$不含有全零列且每一列都不相同。

8.4 有效解码

      如果$H$是一个$m\times{n}$的矩阵并且$x\in\mathbb{Z}_2^n$，那么我们称x的伴随式（syndrome)为$Hx$。
      命题8.36 令$m\times{n}$矩阵$H$决定一个线性编码并且令$x$是一个接收到的n元组。将$x$写成$x=c+e$的形式，其中$c$是传输过来的码字，e是传输的错误。那么x的伴随式$Hx$也是错误$e$的伴随式。
      定理8.37 令$H\in\mathbb{M}_{m\times{n}}(\mathbb{Z}_2)$且假设$H$决定的线性编码是单错纠正的。令$r$是一个接收到的n元组，且最多只会出现1个错误。如果$r$的伴随式为0，那么没有错误出现，否则，如果$r$的伴随式等于$H$的某一列，比如说第i列，那么错误出现在第i位。

陪集译码（Coset Decoding)

      陪集或标准译码使用$C$在$\mathbb{Z}_2^n$中的陪集来完成译码。假设$C$是一个$(n,m)$-线性编码，$C$在$\mathbb{Z}_2^n$中的一个陪集写为$x+C$，其中$x\in\mathbb{Z}_2^n$。根据拉格朗日定理，$C$在$\mathbb{Z}_2^n$中有$2^{n-m}$个不同的陪集。
      命题8.43 令$C$是一个由矩阵$H$给出的$(n,k)$-线性编码，假设$x,y\in\mathbb{Z}_2^n$，那么$x$和$y$在C的相同的陪集中当且仅当$Hx=Hy$，也就是说，两个n元组在相同的陪集中当且仅当它们的伴随式相同。
