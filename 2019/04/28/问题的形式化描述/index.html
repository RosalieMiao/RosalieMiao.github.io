问题的形式化描述

Posted on 2019-04-28 | In 问题求解

JH 2.3 算法的基础

2.3.1 字母表、单词和语言

      定义2.3.1.1 任何非空的，有限的集合都可以被称作一个字母表（alphabet）。一个字母表$\Sigma$的任何一个元素都可以被称作是$\Sigma$的一个符号（symbol）。

      定义2.3.1.2 令$\Sigma$是一个字母表。$\Sigma$上的一个单词（word）是指任意一个有限的$\Sigma$的符号的序列。空单词（empty word）$\lambda$是指唯一的一个不含符号的单词。$\Sigma$上的所有单词的集合用$\Sigma^{\ast}$来表示。

      定义2.3.1.3 字母表$\Sigma$上单词w的长度（length），由$|w|$表示，是指w中字母的个数。对于任意一个单词$w\in\Sigma^{\ast}$以及任意一个字母$a\in\Sigma$，$#_a(w)$是指a在w中出现的次数。

      定义2.3.1.4 令$\Sigma$是一个字母表。那么，对于任意一个$n\in\mathbb{N}$，$\Sigma^n=\{x\in\Sigma^{\ast}||x|=n\}$。

      定义2.3.1.5 对于字母表$\Sigma$上的两个单词v和w，我们定义v和w的拼接（concatenation）为一个由v的符号按顺序排列，后面紧跟着w的符号按顺序排列组成的单词，记为vw，（或者$v\cdot{w}$）。对于任意一个单词$w\in\Sigma^{\ast}$，我们定义
（1） $w^0=\lambda$，并且
（2） $w^{n + 1}=w\cdot{w^n}=ww^n$，对于所有的正整数n。

      一个单词$w\in\Sigma^{\ast}$的前缀（prefix）是指任何一个单词v满足对于某个$\Sigma$上的单词u，有$w=vu$。一个单词$w\in\Sigma^{\ast}$的后缀（suffix）是指任何一个单词u满足对于某个$\Sigma$上的单词x，有$w=xu$。$\Sigma$上的一个单词w的子单词（subword）是指任何一个单词z满足对于某两个$\Sigma$上的单词u，v，有$w=uzv$。

      定义2.3.1.9 令$\Sigma$是一个字母表，每一个子集$L\subseteq\Sigma^{\ast}$都可以被称为一个$\Sigma$上的语言（Language）。关于$\Sigma$的语言L的补充（complement） 是指$L^C=\Sigma^{\ast}-L$。
      令$\Sigma_1$和$\Sigma_2$都是字母表，且令$L_1\subseteq\Sigma^{\ast}_1$，$L_2\subseteq\Sigma^{\ast}_2$都是语言，则$L_1$和$L_2$的拼接（concatenation） 是指$L_1L_2=L_1\circ{L_2}=\{uv\in(\Sigma_1\cup\Sigma_2)^{\ast}|u\in{L_1},v\in{L_2}\}$。

      定义2.3.1.10 令$\Sigma=\{s_1,s_2,\cdots,s_m\}$，$m\ge1$，是一个字母表，且令$s_1<s_2\cdots<s_m$是$\Sigma$上第一个线性顺序。我们定义$\Sigma^{\ast}$上的标准顺序（canonical ordering） 如下。对于所有的$u,v\in\Sigma^{\ast}$，如果$|u|<|v|$或者$|u|=|v|,u=xs_iu’,v=xs_jv’$，对于某些$x,u’,v’\in\Sigma^{\ast}$，且$i<j$，那么$u<v$。

2.3.2 算法问题

      如果一个问题不存在一个已知的确定的算法可以有效的解决它，则我们认为这是一个难问题。有效意味着在低幂的线性时间内可以解决。

      如果$A$是一个算法，$x$是一个输入，则$A(x)$是指算法A对于输入x得到的输出。

      定义2.3.2.1 一个决策问题（decision problem） 是指一个三元组$(L,U,\Sigma)$，其中$\Sigma$是一个字母表且$L\subseteq{U}\subseteq\Sigma^{\ast}$。一个算法A解决（solves,decides） 了决策问题$(L,U,\Sigma)$是指对于所有的$x\in{U}$，
（1）$A(x)=1$ 如果$x\in{L}$，并且
（2）$A(x)=0$ 如果$x\in{U-L}(x\notin{L})$。

      接下来是几个基本的决策问题。

素数测试（PRIMALITY TESTING）

Primality testing
      Input:     An $x\in\Sigma^{\ast}_{bool}$
      Output: “yes” if Number(x) is a prime, “no” otherwise.

多项式等价问题（EQUIVALENCE PROBLEMS FOR POLYNOMIALS）

EQ-POL
      Input:     A prime p, two polynomials $p_1$ and $p_2$ over variables from $X=\{x_1,x_2,\cdots\}$.
      Output: “yes” if $p_1\equiv{p_2}$ in the field $\mathbb{Z}_p$, “no” otherwise.

EQUIVALENCE PROBLEMS FOR ONE-TIME-ONLY BRANCHING PROGRAMS

EQ-1BP
      Input:     One-time-only branching program $B_1$ and $B_2$ over a set of Boolean variables $X=\{x_1,x_2,x_3,\cdots\}$
      Output: “yes” if $B_1$ and $B_2$ are equivalent (represent the same Boolean function), “no” otherwise.

可满足问题（SATISFIABLITY PROBLEM）

      可满足问题是用来决定对于一个给定的合取范式（CNF），这个式子是否可满足。因此，可满足问题是一个决策问题$(SAT,\Sigma_{logic})$，其中$SAT=\{w\in\Sigma_{logic}^+|w$ is a code of a satisfiable formula in CNF $\}$。
      我们也考虑分句个数有限制的合取范式的可满足问题。对于任意正整数$k\ge2$，我们定义k-satisfiablity problem为决策问题$(kSAT,\Sigma_{logic})$，其中$kSAT=\{w\in\Sigma_{logic}^+|w$ is a code of a satisfiable formula in kCNF$\}$。

      接下来我们来定义一些图论方面的可判定性问题。

团问题（CLIQUE PROBLEM）

      团问题是用于决策对于一个给定的图G和正整数k，判断G中是否包含一个大小为k的团。
      团问题是决策问题$(CLIQUE,\{0,1,#\})$，其中$CLIQUE=\{x#w\in\{0,1,#\}^{\ast}|x\in\{0,1\}^{\ast}$ and $w$ represents a graph that contains a clique of size $Number(x)\}$。

点覆盖问题（VERTEX COVER PROBLEM）

      点覆盖问题（VCP）是指决策问题$(VCP,\{0,1,#\})$，其中$VCP=\{u#w\in\{0,1,#\}^+|w\in\{0,1\}^+$ and $w$ represents a graph that contains a vertex cover of size $Number(u)\}$。

哈密尔顿回路问题（HC）

      决策问题$(HC,\{0,1,#\})$，其中$HC=\{w\in\{0,1,#\}^{\ast}|w$ represents a graph that contains a Hamiltonian cycle$\}$

线性规划中存在性问题（EXISTENCE PROBLEMS IN LINEAR PROGRAMMING）

      对于$\mathbb{R}$的任何子集$S$，我们定义$Sol_S(A,b)=\{X\subseteq{S^n}|A\cdot{X}=b\}$
      决策问题$(SOL-IP,\{0,1,#\})$，其中$SOL-IP=\{\langle{A,b}\rangle\in\{0,1,#\}^{\ast}|Sol_Z(A,b)\ne\emptyset\}$。
      0-1线性规划解的存在性问题为决策问题$(SOL-0/1-IP,\{0,1,#\})$，其中$SOL-0/1-IP=\{\langle{A,b}\rangle\in\{0,1,#\}^{\ast}|Sol_{\{0,1\}}(A,b)\ne\emptyset\}$。
      模p线性规划解的存在性问题为决策问题$(SOL-IP_p,\{0,1,\cdots,p-1,#\})$，其中$SOL-IP_p=\{\langle{A,b}\rangle\in\{0,1,\cdots,p-1,#\}^{\ast}|$if A is an $m\times{n}$ matrix over $\mathbb{Z}_p$, $m,n\in\mathbb{N}-\{0\}$, and $b\in\mathbb{Z}_p^m$, then there exists $X\in(\mathbb{Z}_p)^n$ such that $AX\equiv{b}(\mod{p})\}$。

      接下来我们定义几个最优化问题。
      定义2.3.2.2 一个最优化问题（optimization problem） 是指一个7元组$U=(\Sigma_I,\Sigma_O,L,L_I,M,cost,goal)$，其中
（1）$\Sigma_I$是一个字母表，称为U的输入字母表（input alphabet）
（2）$\Sigma_O$是一个字母表，称为U的输出字母表（output alphabet）
（3）$L\subseteq\Sigma_I^{\ast}$是可行的例子的语言（the language of feasible problem instances）
（4）$L_I\subseteq{L}$是U的实际的例子的语言（the language of the (actual) problem instances of U）
（5）$M$是从$L$到$Pot(\Sigma_O^{\ast})$（即$\Sigma_O^{\ast}$的所有子集的集合）的一个函数，对于所有的$x\in{L}$，$M(x)$称为x的可行解集合（the set of feasible solutions）
（6）cost是指消耗方程（cost function），也就是对于任意对$(u,x)$，其中对于某个$x\in{L}$，有$u\in{M(x)}$，分配了一个正实数$cost(u,x)$
（7）$goal\in\{minimum,maximum\}$。
对于任意一个$x\in{L_1}$，对于一个可行解$y\in{M(x)}$，如果$cost(y,x)=goal\{cost(z,x)|z\in{M(x)}\}$，则y被称为对于x和U最优（optimal for x and U）。对于一个最优解$y\in{M(x)}$，我们用$Opt_U(x)$来表示。如果$goal=maximum$，则U被称为最大化问题（maximization problem），如果$goal=minimum$，则U是最小化问题（minimum problem）。定义$Output_U(x)\subseteq{M(x)}$为所有x在U上的最优解的集合。
      如果对于任何$x\in{L_1}$，一个算法A，有$A(x)\in{M(x)}$，则称这个算啊对于U是一致的（consistent）。如果满足以下两点，则称算法B解决了最优化问题U：
（1）B对于U是一致的，并且
（2）对于任意$x\in{L_1}$，$B(x)$是x对于U的最优解。

      定义2.3.2.3 令$U_1=(\Sigma_I,\Sigma_O,L,L_{I,1},M,cost,goal)$和$U_2=(\Sigma_I,\Sigma_O,L.L_{I,2},M,cost,goal)$为两个最优化问题。如果$L_{I,1}\subseteq{L_{I,2}}$，则称$U_1$是$U_2$的子问题（subproblem）。

      我们通过区分以下四点来定义最优化问题：

  • 集合$L_1$
  • 输入的限制
  • 消耗方程
  • 目标

旅行消费者问题（TRAVELING SALESPERSON PROBLEM）

Traveling Salesperson Problem(TSP)
      Input:     A weighted complete graph $(G,c)$, where $G=(V,E)$ and $c:E\to\mathbb{N}$. Let $V=\{v_1,\cdots,v_n\}$ for some $n\in\mathbb{N}-\{0\}$.
      Constraints: For every input instance $(G,c)$, $M(G,c)=\{v_{i_1},v_{i_2},\cdots,v_{i_n},v_{i_1}|(i_1,i_2,\cdots,i_n)$ is a permutation of $(1,2,\cdots,n)\}$,i.e., the set of all Hamiltonian cycles of G.
      Costs:     For every Hamiltonian cycle $H=v_{i_1}v_{i_2}\cdots{v_{i_n}}v_{i_1}\in{M(G,c)}$, $cost((v_{i_1},v_{i_2},
\cdots,v_{i_n},v_{i_1}),(G,c))=\sum_{j=1}^nc(\{v_{i_j},v_{i_{(j\mod{n})+1}}\})$, i.e., thet cost of every Hamiltonian cycle H is the sum of the weights of all edges of H.
      Goal: minimum.

MAKESPAN SCHEDULING PROBLEM

      MS是用于策划那个工作，每个工作都有对应的时间，需要在m个不同的机器上以某种方式执行，并使得时间最短。
Makespan Scheduling Problem(MS)
      Input:     Positive integers $p_1,p_2,\cdots,p_n$ and an integer $m\ge2$ for some $n\in\mathbb{N}-\{0\}$.{$p_i$ is the processing time of the ith jod on any of the m available machines}.
      Constraints: For every input instance $(p_1,\cdots,p_n,m)$ of MS, $M(p_1,\cdots,p_n,m)=\{S_1,S_2,\cdots,S_m|S_i\in\{1,2,\cdots,n\}$ for $i=1,\cdots,m,\cup_{k=1}^mS_k=\{1,2,\cdots,n\}$, and $S_i\cap{S_j}=\emptyset$ for $i\ne{j}\}$.{$M(p_1,\cdots,p_n,m)$ contains all partitions of $\{1,2,\cdots,n\}$ into m subsets. The meaning of $(S_1,S_2,\cdots,S_m)$ is that for $i = 1,\cdots,m$, the jobs with indices from $S_i$ have to be processed on the ith machine}.
      Costs:     For each $(S_1,S_2,\cdots,S_m)\in{M(p_1,\cdots,p_n,m)}$, $cost((S_1,\cdots,S_m),(p_1,\cdots,p_n,m))=max\{\sum_{l\in{S_i}}p_l|i=1,\cdots,m\}$.
      Goal:     minimum.

最小点覆盖（MIN-VCP）

Input: A graph $G=(V,E)$
Constraints: $M(G)=\{S\subseteq{V}|$ every edge of E is incident to at leasr one vertex of S$\}$.
Cost: For every $S\in{M(G)},cost(S,G)=|S|$
Goal: minimum.

集合覆盖问题（SCP）

Input: $(X,F)$, where X is a finite set and $F\subseteq{Pot(X)}$ such that $X=\cup_{S\in{F}}S$.
Constraints: For every input $(X,F)$, $M(X,F)=\{C\subseteq{F}|X=\cup_{S\in{C}}S\}$.
Costs: For every $C\in{M(X,F)},cost(C,(X,F))=|C|$
Goal: minimum

最小权重点覆盖问题（WEIGHT-VCP）

Input: A weighted graph $G=(V,E,c)$,$c:V\to\mathbb{N}-\{0\}$.
Constraints: For every input instance $G=(V,E,c)$,$M(G)=\{S\subseteq{V}|$ S is a vertex cover of G$\}$.
Costs: For every $S\in{M(G)},G=(V,E,c),cost(S,(V,E,c))=\sum_{v\in{S}}c(v)$.
Goal: minimum.

最大团问题（MAX-CL）

Input: A graph $G=(V,E)$
Constraints: $M(G)=\{S\subseteq{V}|\{\{u,v\}|u,v\in{S},u\ne{v}\}\subseteq{E}\}$.{$M(G)$ contains all complete subgraphs (cliques) of G}.
Cost: For every $S\in{M(G)},cost(S,G)=|S|$.
Goal: maximum

最大割问题（MAX-CUT）

Input: A graph $G=(V,E)$
Constraints: $M(G)=\{(V_1,V_2)|V_1\cup{V_2}=V,V_1\ne\emptyset\ne{V_2}$ and $V_1\cap{V_2}=\emptyset\}$.
Costs: For every cut $(V_1,V_2)\in{M(G)}$, $cost((V_1,V_2),G)=|E\cap\{\{u,v\}|u\in{V_1},v\in{V_2}\}|$
Goal: maximum.

简单背包问题（SKP）

Input: A positive integer b, and positive integers $w_1,w_2,\cdots,w_n$ for some $n\in\mathbb{N}-\{0\}$.
Constraints: $M(b,w_1,w_2,\cdots,w_n)=\{T\subseteq\{1,\cdots,n\}|\sum_{i\in{T}}w_i\le{b}\}$
Costs: For each $T\in{M(b,w_1,w_2,\cdots,w_n)}$,$cost(T,b,w_1,w_2,\cdots,w_n)=\sum_{i\in{T}}w_i$.
Goal: maximum.

背包问题（KP）

Input: A positive integer b, and 2n positive integers $w_1,w_2,\cdots,w_n,c_1,c_2,\cdots,c_n$ for some $n\in\mathbb{N}-\{0\}$
Constraints: $M(b,w_1,\cdots,w_n,c_1,\cdots,c_n)=\{T\subseteq\{1,\cdots,n\}|\sum_{i\in{T}}w_i\le{b}\}$.
Costs: For each $T\in{M(b,w_1,\cdots,w_n,c_1,\cdots,c_n)},cost(T,b,w_1,\cdots,w_n,c_1,\cdots,c_n)=\sum_{i\in{T}}c_i$.
Goal: maximum.

01背包问题（BIN-P）

Input: n rational numbers $w_1,w_2,\cdots,w_n\in[0,1]$ for some positive integer n.
Constraints: $M(w_1,w_2,\cdots,w_n)=\{S\subseteq\{0,1\}^n|$ for every $s\in{S}$, $s^T\cdot(w_1,w_2,]cdots,w_n)\le1$, and $\sum_{s\in{S}}s=(1,1,\cdots,1)\}$.
Costs: For every $S\in{M(w_1,w_2,\cdots,w_n)},cost(S,(w_1,\cdots,w_n))=|S|$.
Goal: minimum.

最大可满足问题（MAX-SAT）

Input: A formula $\Phi=F_1\land{F_2}\land\cdots\land{F_m}$ over $X=\{x_1,x_2,\cdots\}$ in CNF
Constraints: For every formula $Phi$ over the set $\{x_1,\cdots,x_n\}\subseteq{X}$, $n\in\mathbb{N}-\{0\}$, $M(\Phi)=\{0,1\}^n$
Costs: For every $Phi$ in CNF, and every $\alpha\in{M(\Phi)}$, $cost(\alpha,\Phi)$ is the number of clauses satisfied by $\alpha$.
Goal: maximum.

线性规划（LP）

Input: A matrix $A=[a_{ij}]_{i=1,\cdots,m,j=1,\cdots,n}$, a vector $b\in\mathbb{R}^m$, and a vector $c\in\mathbb{R}^n$, $n,m\in\mathbb{N}-\{0\}$.
Constraints: $M(A,b,c)=\{X\in\mathbb{R}^n|A\cdot{X}=b$ and the elements of X are non-negtive reals only$\}$.
Costs: For every $X=(x_1,\cdots,x_n)\in{M(A,b,c)},c=(c_1,\cdots,c_n)^T,cost(X,(A,b,c))=c^T\cdot{X}=\sum_{i=1}^nc_ix_i$
Goal: minimum

整数线性规划（IP）

Input: An $m\times{n}$ matrix $A=[a_{ij}]_{i=1,\cdots,m,j=1,\cdots,n}$ and two vectors $b=(b_1,\cdots,b_m)^T,c=(c_1,\cdots,c_n)^T$ for some $n,m\in\mathbb{N}-\{0\},a_ij,b_i,c_j$ are integers for $i=1,\cdots,m,j=1,\cdots,n$
Constraints: $M(A,b,c)=\{X=(x_1,\cdots,x_n)\in\mathbb{Z}^n|AX=b$ and $x_i\ge0$ for $i=1,\cdots,n\}$
Costs: For every $X=(x_1,\cdots,x_n)\in{M(A,b,c)},cost(X,(A,b,c))=\sum_{i=1}^nc_ix_i$.
Goal: minimum.

最大模k线性方程组问题（MAX-LINMODk）

Input: A set S of m linear equations over n unkowns, $n,m\in\mathbb{N}-\{0\}$, with coefficients from $\mathbb{Z}_k$
Constraints: $M(S)=\mathbb{Z}_k^m$
Costs: For every $X\in{M(S)},cost(X,S)$ is the number of linear equations of S satisfied by X
Goal: maximum.

2.3.3 复杂度理论（Complexity Theory）

      定义2.3.3.1 令$\Sigma_I$和$\Sigma_O$都是字母表，令$A$是实现从$\Sigma_I^{\ast}$到$\Sigma_O^{\ast}$映射的算法。对于任意$x\in\Sigma_I^{\ast}$，$Time_A(x)$用于表示对于输入x，算法A在计算时所需要的时间复杂度，$Space_A(x)$用于表示对于输入x，算法A在计算时所需要的空间复杂度。

      定义2.3.3.2 令$\Sigma_I$和$\Sigma_O$是两个字母表，令$A$是实现从$\Sigma_I^{\ast}$到$\Sigma_O^{\ast}$映射的算法。A的最坏时间复杂度是一个方程$Time_A:(\mathbb{N}-\{0\})\to\mathbb{N}$，$Time_A(n)=max\{Time_A(x)|x\in\sum_I^n\}$对于所有正整数n。A的最坏空间复杂度是一个方程$Space_A:(\mathbb{N}-\{0\})\to\mathbb{N}$，$Space_A(n)=max\{Space_A(x)|x\in\sum_I^n\}$对于所有正整数n。

      定理2.3.3.3 存在一个决策问题$(L,\sum_{bool})$满足对于任何一个算法$A$，存在另外一个算法B，有$Time_B(n)=\log_2(Time_A(n))$，对于无限多个正整数n。

      定义2.3.3.4 令U是一个算法问题，令$f,g$是$\mathbb{N}$到$\mathbb{R}^+$的函数。我们称$O(g(N))$是U的时间复杂度的上限，如果存在一个算法A可以解决U且$Time_A(n)\in{O(g(n))}$。
我们称$\Omega(f(n))$是U的时间复杂度的下限如果对于所有解决U的算法B都有$Time_B(n)\in\Omega(f(n))$。
一个算法C被称为最优的解决U的算法如果$Time_C(n)\in{O(g(n))}$且$\Omega(g(n))$是U的时间复杂度的下限。
