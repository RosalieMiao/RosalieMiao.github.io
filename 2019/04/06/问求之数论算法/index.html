TJ第31章笔记

      输入规模和算术计算的代价
      在本章中，“大输入”通常指包含“大整数”的输入，而不是包含“很多整数”的输入（像排序问题中的那样）。因此，我们利用输入所需的位数来度量输入的大小，而不仅仅是输入中整数的数目。给定k个整数输入$a_1,a_2,\cdots,a_k$，如果算法可以在关于$\lg{a_1}, \lg{a_2}, \cdots, \lg_{a_k}$的多项式时间内完成，即算法在关于二进制编码后的输入长度的多项式时间内完成，则该算法称为多项式时间算法。
      用数论算法所需的位运算数目作为基准来衡量算法的时间代价。在此模型中，讲两个$\beta$位整数用常规方法相乘需要$\Theta(\beta^2)$次位运算。

31.1 基础数论概念

整除性与约数

      符号$d\mid{a}$（读作“d整除a”）的含义是，存在某个整数k，使得$a=kd$。任何整数均可整除0.如果$a>0$且$d\mid{a}$，那么$|d|\le|a|$。如果$d\mid{a}$，则称a是d的倍数。如果d不能整除a，则写作$d\nmid{a}$。
      如果$d\mid{a}$且$d\ge0$，则称d是a的约数。注意，$d\mid{a}$当且仅当$-d\mid{a}$，即a的任何约数的负数同样可以整除a。因此，不失一般性，可规定约束位非负数。非零整数a的约数应至少为1，且不会大于$|a|$。
      任何正整数a均可被平凡约数1和其自身a所整除.整数a的非平凡约数统称为a的因子。

素数与合数

      如果一个整数$a>1$且只能被平凡约束1和它自身所整除，则这个数是素数。
      如果一个整数$a>1$且不是素数，则称之为合数。称整数1为基本单位，并且它既不是素数也不是合数。同样，整数0和所有负整数既不是素数也不是合数。

除法定理、余数和等模

      定理31.1（除法定理） 对于任何整数a和任何正整数n，存在唯一整数q和r，满足$0\le{r}<n$且$a=qn+r$。
      称$q=\lfloor{a/n}\rfloor$为除法的商，值$r=a\mod{n}$为除法的余数。$n\mid{a}$当且仅当$a\mod{n}=0$。

公约数与最大公约数

      如果d是a的约数并且d也是b的约数，则d是a与b的公约数。需要注意的是，1是任意两个整数的公约数。
      公约数的一条重要性质是：$d\mid{a}$且$d\mid{b}$蕴涵着$d\mid(a+b)$且$d\mid(a-b)$。更一般地，对任意整数x和y有$d\mid{a}$且$d\mid{b}$蕴涵着$d\mid(ax+by)$。并且，如果$a\mid{b}$，那么$|a|\le|b|$，或者$b=0$，而这说明$a\mid{b}$且$b\mid{a}$蕴涵着$a=\pm{b}$。
      两个不同时为0的整数a与b的公约数中最大的称为其最大公约数，记作$\gcd(a,b)$。

      定理31.2 如果任意整数a和b不都为0，则$\gcd(a,b)$是a与b的线性组合集$\{ax+by:x,y\in\mathbb{Z}\}$中的最小正元素。
      推论31.3 对任意整数a与b，如果$d\mid{a}$且$d\mid{b}$，则$d\mid\gcd(a,b)$。
      推论31.4 对所有整数a和b以及任意非负整数n，有$\gcd(an,bn)=n\gcd(a,b)$。
      推论31.5 对于任意正整数n、a和b，如果$n\mid{ab}$且$\gcd(a,n)=1$，则$n\mid{b}$。

互质数

      如果两个整数a与b只有公约数1，即$\gcd(a,b)=1$，则a与b称为互质数。
      定理31.6 对任意整数a、b和p，如果$\gcd(a,p)=1$且$\gcd(b,p)=1$，则$\gcd(ab,p)=1$。
      对于整数$n_1,n_2,\cdots,n_k$，如果对任何$i\ne{j}$都有$\gcd(n_i,n_j)=1$，则称整数$n_1,n_2,\cdots,n_k$两两互质。

31.2 最大公约数

      原则上讲，可以根据a和b的素因子分解求出正整数a和b的最大公约数$\gcd(a,b)$。但是目前已知的最好的因子分解算法也不能达到多项式运行时间，因此，利用这种方法来计算最大公约数不太可能获得高效率。
      定理31.9（GCD递归定理） 对任意非负整数a和任意正整数b，$\gcd(a,b)=\gcd(b,a\mod{b})$。

欧几里得算法的运行时间

      引理31.10 如果$a>b\ge1$并且EUCLID(a,b)执行了$k\ge1$次递归调用，则$a\ge{F_{k+2}},b\ge{F_{k+1}}$。
      定理31.11（Lame定理） 对任意整数$k\ge1$。如果$a>b\ge1$，且$b<F_{k+1}$，EUCLID(a,b)的递归调用次数少于k次。
      过程$EUCLID(a,b)$执行中递归调用的次数为$O(\lg{b})$，如果过程EUCLID作用于两个$\beta$位数，则它将执行$O(\beta)$次算术运算和$O(\beta^3)$次位操作（假设$\beta$位数的乘法和除法运算要执行$O(\beta^2)$次位操作）。

31.3 模运算

      可以把模运算非正式地与通常的整数运算一样看待，如果执行模n运算，则每个结果值x都由集合$\{0,1,\cdots,n-1\}$中的某个元素所取代，该元素在模n的意义下与x等价（即用$x\mod{n}$来取代x）。如果仅限于运用加法、减法和乘法运算，则用这样的非正式模型就足够了。下面给出更为形式化的模型。

有限群

      群（$S,\oplus$）是一个集合S和定义在S上的二进制运算$\oplus$，该运算满足下列性质：

  1. 封闭性：对所有$a,b\in{S}$，有$a\oplus{b}\in{S}$。
  2. 单位元：存在一个元素$e\in{S}$，称为群的单位元，满足对所有$a\in{S},e\oplus{a}=a\oplus{e}=a$。
  3. 结合律：对所有$a,b,c\in{S}$，有$(a\oplus{b})\oplus{c}=a\oplus(b\oplus{c})$。
  4. 逆元：对每个$a\in{S}$，存在唯一的元素$b\in{S}$，称为a的逆元，满足$a\oplus{b}=b\oplus{a}=e$。
          如果群$(S,\oplus)$满足交换律，即对所有$a,b\in{S}$，有$a\oplus{b}=b\oplus{a}$，则它是一个交换群。如果群$(S,\oplus)$满足$|S|<\infty$，则它是一个有限群。

子群

      如果$(S,\oplus)$是一个群，$S’\subseteq{S}$，并且$(S’,\oplus)$也是一个群，则$(S’,\oplus)$称为$(S,\oplus)$的子群。
      定理31.14（一个有限群的非空封闭子集是一个子群）如果$(S,\oplus)$是一个有限群，$S’$是S的任意一个非空子集并满足对所有$a,b\in{S’}$，有$a\oplus{b}\in{S’}$，则$(S’,\oplus)$是$(S,\oplus)$的一个子群。
      定理31.15（拉格朗日定理） 如果$(S,\oplus)$是一个有限群，$(S’,\oplus)$是$(S,\oplus)$的一个子群，则$|S’|$是$|S|$的一个约数。
      对一个群S的子群S’，如果$S’\ne{S}$，则子群S’称为群S的真子群。
      推论31.16 如果S’是有限群S的真子群，则$|S’|\le|S|/2$。

由一个元素生成的子群

      由a生成的子群用$\langle{a}\rangle$或$(\langle{a}\rangle,\oplus)$来表示，其定义如下：$\langle{a}\rangle=\{a^{(k)}:k\ge1\}$。我们称a生成子群$\langle{a}\rangle$，或者a是$\langle{a}\rangle$的生成元。
      定理31.17 对任意有限群$(S,\oplus)$和任意$a\in{S}$，一个元素的阶等于它所生成子群的规模，即$ord(a)=|\langle{a}\rangle|$。
      推论31.18 序列$a^{(1)},a^{(2)},\cdots$是周期序列，其周期为$t=ord(a)$，即$a^{(i)}=a^{(j)}$当且仅当$i\equiv{j(\mod{t})}$。
      推论31.19 如果$(S,\oplus)$是具有单位元e的有限群，则对所有$a\in{S},a^{|S|}=e$。

31.4 求解模线性方程

      现在来考虑求解下列方程的问题：$ax\equiv{b(\mod{n})}$，其中$a>0,n>0$。
      定理31.20 对任意正整数a和n，如果$d=\gcd(a,n)$，则在$\mathbb{Z}_n$中，$\langle{a}\rangle=\langle{b}\rangle=\{0,d,2d,\cdots,((n/d)-1)d\}$。因此$|\langle{a}\rangle|=n/d$。
      推论31.21 当且仅当$d\mid{b}$时，方程$ax\equiv{b(\mod{n})}$对于未知量x有解，这里$d=\gcd(a,n)$。
      推论31.22 方程$ax\equiv{b(\mod{n})}$要么对模n有d个不同的解，要么无解，这里$d=\gcd(a,n)$。
      定理31.23 令$d=\gcd(a,n)$，假设对某些整数x’和y’，有$d=ax’+ny’$（例如EXTENDED-EUCLID所计算出的结果）。如果$d\mid{b}$，则方程$ax\equiv{b(\mod{n})}$有一个解的值为$x_0$，这里$x_0=x’(b/d)\mod{n}$。
      定理31.24 假设方程$ax\equiv{b(\mod{n})}$有解，（即$d\mid{b}$，这里$d=\gcd(a,n)$），且$x_0$是该方程的任意一个解。因此，该方程对模n恰有d个不同的解，分别为$x_i=x_0+i(n/d)$，这里$i=0,1,\cdots,d-1$。
      现在已经为求解方程$ax\equiv{b(\mod{n})}$完成了数学上的必要准备，下列算法可输出该方程的所有解。输入a和n为任意正整数，b为任意整数。
MODULAR-LINEAR-EQUATION-SOLVER(a,b,n)
1  (d,x’,y’)=EXTENDED-EUCLID(a,n)
2  if $d\mid{b}$
3        $x_0=x’(b/d)\mod{n}$
4        for i=0 to d-1
5              print$(x_0+i(n/d))\mod{n}$
6  else print “no solutions”

      推论31.25 对任意$n>q$，如果$\gcd(a,n)=1$，则方程$ax\equiv{b}(\mod{n})$对模n有唯一解。
      推论31.26 对任意$n>1$，如果$\gcd(a,n)=1$，那么方程$ax\equiv1(\mod{n})$对模n有唯一解；否则方程无解。

31.5 中国余数定理

31.6 元素的幂

      定理31.30（欧拉定理） 对于任意整数$n>1$，$a^{\phi(n)}=1(\mod{n})$对所有$a\in\mathbb{Z}_n^{\ast}$都成立。
      定理31.31（费马定理） 如果$p$是素数，则$a^{p-1}=1(\mod{n})$对所有$a\in\mathbb{Z}_n^{\ast}$都成立。
      如果$ord_n(g)=|\mathbb{Z}_n^{\ast}|$，则对模n，$\mathbb{Z}_n^{\ast}$中的每个元素都是g的一个幂，且g是$\mathbb{Z}_n^{\ast}$的一个原根或生成元。如果$\mathbb{Z}_n^{\ast}$包含一个原根，就称群$\mathbb{Z}_n^{\ast}$是循环的。
      定理31.32 对所有的素数$p>2$和所有正整数e，使得$\mathbb{Z}_n^{\ast}$是循环群的$n>1$的值为$2,4,p^e,2p^e$。
      如果g是$\mathbb{Z}_n^{\ast}$的一个原根且a是$\mathbb{Z}_n^{\ast}$中的任意元素，则存在一个z，使得$g^z=a(\mod{n})$。这个z称为对模n到基g上的a的一个离散对数或指数，将这个值表示为$ind_{n,g}(a)$。
      定理31.33（离散对数定理） 如果g是$\mathbb{Z}_n^{\ast}$的一个原根，则当且仅当等式$x\equiv{y(\mod\phi(n))}$成立时，有等式$g^x\equiv{g^y}(\mod{n})$成立。
      定理31.34 如果p是一个奇素数且$e\ge1$，则方程$x^2\equiv1(\mod{p^e})$仅有两个解，即$x=1$和$x=-1$。
      如果一个数x满足方程$x^2\equiv1(\mod{n})$，但x不等于以n为模的1的两个“平凡”平方根：1或-1，则x是一个以n为模的1的非平凡平方根。
      推论31.35 如果对模n存在1的非平凡平方根，则n是合数。

用反复平方法求数的幂

      数论计算中经常出现一种运算，就是求一个数的幂对另一个数的模运算，也称为模取幂。更明确地说，希望有一种高效的方法来计算$a^b\mod{n}$的值，其中a，b为非负整数，n是一个正整数。采用b的二进制表示，反复平方法可以高效地解决这个问题。
      设$\langle{b_k,b_{k-1},\cdots,b_1,b_0}\rangle$是b的二进制表示（即二进制表示有k+1位场，$b_k$位最高有效位，$b_0$为最低有效位。
MODULAR-EXPONENTIATION(a,b,n)
1  c = 0
2  d = 1
3  let $\langle{b_k,b_{k-1},\cdots,b_1,b_0}\rangle$ be the binary representation of b
4  for i=k downto 0
5        c = 2c
6        $d = (d\cdot{d})\mod{n}$
7        if $b_i==1$
8              c = c + 1
9              $d = (d\cdot a)\mod{n}$
10  return d

      如果输入a,b与n都是$\beta$位的数，则需要的算术运算的总次数是$O(\beta)$，并且需要的位操作的总次数是$O(\beta^3)$。
