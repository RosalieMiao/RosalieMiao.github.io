3.2 定义与例子

      整数模n和三角形或矩形对称置换是群的例子。
      一个在集合G上的二元运算（binary operation）或者构成规则（law of composition）是一个函数$G\times{G}\to{G}$，这个函数给每一个二元组$(a,b)\in{G\times{G}}$一个对应的G中的值$a\circ{b}$或者$ab$。$a\circ{b}$或者$ab$称为a和b的组合（composition）。
      一个群(group)$(G,\circ)$是指一个集合以及一个二元运算$(a,b)\mapsto{a\circ{b}}$且这个二元运算满足一下公理(axioms)：

  • 结合律(The law of composition is associative.)
  • 有单位元(There exists an element $e\in{G}$, called the identity element, such that for any element $a\in{G}$)
  • 每个元素都有逆元(For each element $a\in{G}$, there exists an inverse element in G, denoted by $a^{-1}$, such that)      有交换律（对所有的$a,b\in{G}$, $a\circ{b}=b\circ{a}$）的群称为阿贝尔群(abelian)或可交换的(commutative)，否则称为不可交换的(nonabelian or noncommutative)。不是每个群都有交换律。
          群的一种方便的表达方式是以加法或乘法表的方式。一种这样的表称为凯莱表(Cayley table)。
          整数集$Z$上的加法构成阿贝尔群。$Z_n$上的加法也能构成群。$Z_n$上的乘法不能构成群，乘法作用在$Z_n$上时，1为单位元，只有与$n$互质的元素才有逆元素。这些元素组成集合$U(n)$，$U(n)$是一个群，称为$Z_n$的单位群(group of units)。
          等边三角形上的对称置换构成非阿贝尔群。可逆的$2\times2$矩阵构成的集合上的矩阵乘法运算构成非阿贝尔群。一个四元群(quaternion group)的例子。不含0的复数集上的乘法构成群。
          当群中元素的个数是有限个的时候，称这个群是有限的。$|G|$表示群G中元素的个数。

群的基本特征

    命题 3.17 （单位元唯一。）The identity element in a group G is unique; that is, there exists only one element $e\in{G}$ such that $eg=ge=g$ for all $g\in{G}$.
    命题 3.18 （逆元唯一。）If g is any element in a group G, then the inverse of g, debited by $g^{-1}$, is unique.
    命题 3.19 Let G be a group. If $a,b\in{G}$, then $(ab)^{-1}=b^{-1}a^{-1}$.
    命题 3.20 Let G be a group. For any $a\in{G}$, $(a^{-1})^{-1}=a$.
    命题 3.21 Let G be a group and a and b be any two elements in G. Then the equations $ax=b$ and $xa=b$ have unique solutions in G.
    命题 3.22 If G is a group and $a,b,c\in{G}$, then $ba=ca$ implies $b=c$ and $ab=ac$ implies $b=c$.
      right and left cancellation laws(左右消元法则？？？) are true in groups.
    定理 3.23 （指数运算）In a group, the usual laws of exponents hold; that is, for all $g,h\in{G}$,

  1. $g^mg^n=g^{m+n}$ for all $m,n\in{Z}$;
  2. $(g^m)^n=g^{mn}$ for all $m,n\in{Z}$;
  3. $(gh)^n=(h^{-1}g^{-1})^{-n}$ for all $n\in{Z}$. Furthermore, if G is abelian, then $(gh)^n=g^nh^n$.

3.3 子群

定义与例子

      偶数集在加法下构成的群是整数集在加法下构成的群的子群。
      子群的定义：如果H是群G的一个自己并且当G的群运算作用在H上时，H也是一个群，那么H称为是G的子群。
      G的子群$H=\{e\}$称为是平凡子群(trivial subgroup)。一个子群如果是G的真子集，那这个子群称为真子群(proper subgroup)。
      不含0的有理数集在乘法下构成的群是不含0的实数集在乘法下构成的群的子群。
      特殊线性群(special linear group)。
      如果H是G的子集且是群，那么H不一定是G的子群，因为两个群的二元运算有可能不一样。
      可以用子群来判断两个群是否等价。

一些子群的定理

    命题 3.30 G的子集H是子群当且仅当其满足以下条件。

  1. G的单位元e在H中。
  2. （封闭性）如果$h_1,h_2\in{H}$，那么$h_1h_2\in{H}$。
  3. （有逆元）如果$h\in{H}$，那么$h^{-1}\in{H}$。

    命题 3.31 如果H是G的子集，那么H是G的子群当且仅当$H\neq\emptyset$并且任给$g,h\in{H}$，有$gh^{-1}\in{H}$。

4.1 循环子群

      一般地，一个子群完全依赖于群中的一个元素。
    定理 4.3 如果G是一个群并且a是G中的任意一个元素，那么集合

是G的一个子群。更进一步地，$\langle{a}\rangle$是G中包含a的最小的子群。
    注 4.4 如果我们用的是“+”记号的话，比如说整数集在加法下，我们这样写$\langle{a}\rangle$：$\langle{a}\rangle=\{na:n\in{Z}\}$.
      对所有的$a\in{G}$，我们称$\langle{a}\rangle$为由a生成的循环子群(cyclic subgroup)。如果G包含一个元素a并且$G=\langle{a}\rangle$，那么G是一个循环群(cyclic group)。在这个情况下a称为G的生成器(generator)。如果a是G的一个元素，我们定义a的阶数(order)为一个最小正整数n，使得$a^n=e$，写为$|a|=n$。如果不存在这样的n，我们称a是无穷阶的，写为$|a|=\infty$来表示a的阶数。
      1和5可以生成$Z_6$，所以$Z_6$是一个循环群。2在$Z_6$中的阶数是3，由2生成的循环子群为$\langle2\rangle=\{0,2,4\}$。
      Z是循环群，生成器为1或-1.$Z_n$是循环群，1是$Z_n$的生成器，$Z_n$还可能有其他生成器。
      U(9)是一个循环群，生成器为2.（$U(9)=\{1,2,4,5,7,8\}$）
      不是所有的群都是循环群，三角形上的对称置换就不是循环群。
    定理 4.9 每一个循环群都是阿贝尔群。

循环群的子群

    定理 4.10 循环群的每一个子群都是循环的。

    命题 4.12 如果G是一个阶数为n的循环群，假设a是G的生成器，那么$a^k=e$当且仅当n整除k。
    定理 4.13 如果G是一个阶数为n的循环群，假设$a\in{G}$是G的一个生成器。如果$b=a^k$，那么b的阶数是$n/d$，其中$d=gcd(k,n)$。
